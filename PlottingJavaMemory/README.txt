Bevor das Skript main.py gestartet werden kann, muss man zuerst alle Bibliotheken herunterladen. Hierfuer kann man den folgenden Befehl verwenden:
pip install -r requirements.txt


Um dieses Programm zu starten, verwenden Sie den folgenden Befehl:
python main.py oder python3 main.py 

Die CSV-Dateien wurden aus der JConsole exportiert.
